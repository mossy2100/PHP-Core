10/**
11 * This class provides a method of formatting any PHP value as a string, with a few differences from the default

13 *

17 * 4. Resources are also encoded in a style similar to HTML tags.
18 *
19 * The purpose of the class is to offer a somewhat more concise, readable, and informative alternative to the usual
